## 生命周期

```ts
onBeforeMount, // 创建之前，获取不到dom元素
onMounted, // dom元素渲染完成
onBeforeUpdate, // 更新之前
onUpdated, // 更新之后
onBeforeUnmount, // 卸载之前
onUnmounted, // 卸载完成
```
